Features of Store

Speed . Ease . Handy

Ubuntu Play Store is very fast and low in memory consumption it will take approx 3 Mb after install .

It take only two seconds to launch i.e, 500% faster then Ubuntu software center.

It takes one Second to grab the list of software from my repository .

Every time you will launch you will get updated list of apps i.e, you dont need to run “sudo apt-get update” , it will get new and updated list of software including every minute change done by developer to their apps over the night .

Always Up to Date

Instant Client and Applications Updates

Ubuntu Play Store provide a unique way to update itself so you just click few buttons from your mouse and get a always up to date store bundled with new features and support.

Applications inside are easy to update , we keep both developers and users in mind , so we made a flexible system for both developers and users so they can update their packages easily with few clicks.

Client loads always a up to date list of softwares , and informations from server with this unique feature of store users never need to do "sudo apt-get update" to get a updated list of software information.
